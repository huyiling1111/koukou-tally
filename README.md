# 抠抠记账

一款基于 Vue 技术栈开发的简介本地记账 WebApp

# 技术栈

- Vue
- Vuex
- VueRouter
- TypeScript
- SCSS

# 功能

抠抠记账是一款简化后的记账应用，也是一款基于 Vue、Vuex、Vue-Router、TypeScript 的单页面应用，
所有需求由本人独立实现。

记录账单（包括金额、日期、分类等）  
分类编辑（增、改）
每日金额明细
近 30 天图表展示
